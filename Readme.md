# VehicleBay - Vehicle Service Management System

**VehicleBay** is a comprehensive web application designed to streamline vehicle service operations. This project was developed as a student group initiative to bridge the gap between theoretical database management systems (DBMS) and real-world application implementation.

---

## 🚀 Project Overview
The primary goal of VehicleBay is to provide a robust platform for managing service appointments, tracking vehicle history, and organizing customer data efficiently. By integrating a relational database with a modern web stack, we focused on data integrity, efficient querying, and a seamless user experience.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
* **Templating Engine:** EJS (Embedded JavaScript)
* **Backend:** Node.js, Express.js
* **Database:** PostgreSQL

## 🌟 Key Features
* **Service Scheduling:** Efficiently book and manage vehicle maintenance slots.
* **Relational Data Management:** Organized storage for customers, vehicles, and service logs using PostgreSQL.
* **Responsive Design:** Fully accessible across desktop and mobile devices thanks to Bootstrap 5.
* **Dynamic Rendering:** Real-time data injection using EJS for a personalized user dashboard.
